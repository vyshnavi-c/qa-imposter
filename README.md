# QA Imposter

A QA-themed social deduction word game for teams. One player gets a different (but related) QA word — can the group find who it is?

## How to play

1. Add all player names and pick a word category
2. Pass the device around — each player taps privately to see their secret word
3. Everyone gives one verbal clue about their word (no saying the word itself)
4. Vote for who you think the imposter is
5. See if the team caught them!

## Deploy

This site is deployed automatically to GitHub Pages on every push to `main` via GitHub Actions.

To set it up:
1. Push this repo to GitHub
2. Go to **Settings → Pages → Source → GitHub Actions**
3. Push to `main` — the workflow deploys automatically
4. Share the URL: `https://<your-username>.github.io/<repo-name>/`
